# Examples for yc and terraform (Yandex Cloud)

## Install YC cli
```
curl https://storage.yandexcloud.net/yandexcloud-yc/install.sh | bash
```

## Init YC cli
```
yc init
```

## Get token, cloud-id, folder-id
```
yc config list
```
Output:
```
token: xxx
cloud-id: xxx
folder-id: xxxx
compute-default-zone: ru-central1-b
```
